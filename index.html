Heuristic Evaluation Report

(HCI2) Human-Computer Interaction 2 - Final Project

Evaluated System

Library Management System

This heuristic evaluation was conducted on Valladolid NHS Library Management System, a web application designed to manage book inventory, track borrowed and returned books, and assist in library administration for Valladolid National High School.

The evaluation was performed using Jakob Nielsen's 10 Usability Heuristics and the DECIDE framework to identify usability issues and propose improvements.
